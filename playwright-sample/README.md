# Playwright のサンプル

## 環境構築

- 事前準備
  - node.js と yarn の install
    - https://qiita.com/sefoo0104/items/0653c935ea4a4db9dc2b
    - https://qiita.com/suisui654/items/1b89446e03991c7c2c3d
- ライブラリとブラウザのインストール
  ```bash
  # ライブラリインストール
  $ yarn
  # ブラウザインストール
  $ npx playwright install chromium
  ```

## テスト実行

```bash
$ yarn test
```
